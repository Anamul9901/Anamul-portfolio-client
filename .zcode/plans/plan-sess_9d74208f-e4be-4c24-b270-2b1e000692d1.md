## Phase 1 ŌĆö New design system: "Ember Editorial"

A deliberate visual shift away from the current mint, applied everywhere via CSS variables (so it's trivially swappable later).

**New tokens (`src/styles/globals.css`):**
- Base ink: `--bg-0 #08090B`, `--bg-1 #101218`, `--bg-2 #161A22`
- Hairlines: `rgba(255,255,255,0.08)` / strong `0.14`
- Text: `--text-0 #F5F6F8`, `--text-1 #A3A8B3`, `--text-2 #6B7280`
- **Accent (new): `--accent #F59E0B` (ember/amber)**, `--accent-soft rgba(245,158,11,0.12)`, `--accent-glow-soft rgba(245,158,11,0.22)` ŌĆö warm, premium, editorial, a clear visual change from mint. Used sparingly on section indices, hover states, underlines, dots.
- **Strict spacing scale (8px base)** ŌĆö fixes the "perfect spacing" ask: sections `py-24 md:py-32` (was inconsistent `py-20 md:py-28`), section-header `mb-14 md:mb-16`, grid gaps `gap-6 md:gap-8`. One rhythm across the whole site.
- Type scale: display `clamp(2.75rem,6vw,5rem)`, section heading `clamp(2rem,3.2vw,3rem)`. Inter (display/body) + Fira Code (mono labels).

Because everything already reads from `var(--accent)` etc., most sections re-skin automatically; I'll then refine spacing/motion per section.
